Choose a file and click the "cola1 layout" button a few times to layout the topology. You can also help between layouts by pulling nodes around by hand. Try the different variants of cola, as well.

For files ending in _graph, left-click on a node to select it as a destination and see the shortest path tree (green), as well as MRT red and blue trees for that destination.

Right-click on a node to simulate failing that node and see the MRT alternates selected by adjacent nodes to reach the destination (shown with dashed arrows)

Hide or show different tree and graph components using the checkboxes.

Some topologies have nodes numbered 2000 and above. These are named proxy nodes representing multi-homed prefixes.
