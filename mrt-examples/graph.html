This was tested with FireFox 31.1 and 40.2. Others browsers should work, but they have not been tested.













Choose below what left-clicking on a node does:

selects destination
selects failed node
selects GADAG root
toggles visibility of edges to a prefix

For more info on Maximally Redundant Trees(MRTs), see MRT algorithm

Choose a topology and click the layout buttons. You can also pull nodes around by hand.

Select a destination to see MRT-red, MRT-blue, and green (SPF) trees for that destination.

Right-click on a node to simulate failing that node and see the MRT alternates selected by adjacent nodes to reach the destination (shown with dashed arrows)

Hide or show different tree and graph components using the checkboxes.

Some topologies have nodes numbered 2000 and above. These are named proxy nodes representing multi-homed prefixes.
